- `cdn_waf`: Detected protection layers.

- `dns_records`: IP resolution details.

### ANALYSIS LOGIC

3. **Red Team Prioritization**: 
   - High Priority: Dev/Staging environments, Admin portals, API gateways without WAF.
   - Low Priority: Static assets, parked domains, heavy CDN/WAF protection.
4. **Vulnerability Correlation**: If `existing_vulnerabilities` is not empty, suggest how to escalate those specific findings.
