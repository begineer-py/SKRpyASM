# War Stories — Web 安全實戰經驗庫

真實滲透測試 / CTF 解題戰報。

每份戰報記錄完整的攻擊鏈：從信息收集到最終 flag，包括走了哪些彎路、關鍵突破口在哪裡。

## 文件命名規則

```
YYYY-MM-DD_題目類型_關鍵詞.md
```

例如：`2026-04-19_php-deserialization_regex-bypass.md`

## 戰報模板

每份戰報應包含：

| 區塊 | 內容 |
|------|------|
| **元信息** | 日期、目標、類型、關鍵詞、輪數、工具鏈 |
| **攻擊鏈** | 每一步做了什麼、發現了什麼 |
| **關鍵突破** | 哪一步是決定性的，為什麼 |
| **走的彎路** | 哪些嘗試失敗了，原因是什麼 |
| **Payload** | 最終可復現的利用代碼 |
| **經驗總結** | 可遷移到同類題目的方法論 |
